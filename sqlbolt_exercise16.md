## Create a new table named `Database`

The table should have the following columns: - **Name**: A string (text)
describing the name of the database - **Version**: A floating-point
number representing the latest version - **Download_count**: An integer
representing how many times the database was downloaded

**Query:**

``` sql
CREATE TABLE Database (
    Name TEXT,
    Version FLOAT,
    Download_count INTEGER
);

```
